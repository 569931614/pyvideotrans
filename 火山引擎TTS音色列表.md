## 如何选择音色

### 在 GUI 中选择

1. 启动程序: `python sp.py`

2. 在主界面:
   - **TTS类型**: 选择 **"字节火山TTS"**
   - **配音角色**: 会自动显示上述所有音色

3. 点击角色下拉框,选择您喜欢的音色

## Cluster ID 说明

### 什么是 Cluster ID?

**定义**: Cluster ID 是火山引擎 TTS 服务的集群标识符,用于指定使用哪个服务实例。

### 常见值

| Cluster ID | 说明 | 使用场景 |
|-----------|------|---------|
| `volcengine_streaming` | 流式合成集群 | 实时语音合成(推荐) |
| `volcano_tts` | 批量合成集群 | 批量离线合成 |
| 自定义集群ID | 专属集群 | 大客户专用 |

### 如何获取

1. 登录火山引擎控制台: https://console.volcengine.com/speech/app

2. 进入您的应用详情

3. 查看 **"集群信息"** 或 **"接入配置"** 部分

4. 复制 Cluster ID

### 新手建议

如果不确定,可以:
1. 留空(系统会使用默认集群)
2. 使用 `volcengine_streaming`
3. 咨询火山引擎客服

## 完整配置示例

### 配置文件 (videotrans/params.json)

```json
{
  "volcenginetts_appid": "您的App ID",
  "volcenginetts_access": "您的Access Token",
  "volcenginetts_cluster": "volcengine_streaming"
}
```

### 配音流程

1. **配置火山TTS** (GUI或手动)
   ```
   设置 → 配音服务 → 字节火山TTS
   ```

2. **选择视频和语言**
   - 源语言: 例如 `中文`
   - 目标语言: 例如 `英文`

3. **选择TTS和音色**
   - TTS类型: `字节火山TTS`
   - 配音角色: 从下拉列表选择,例如 `灿灿2.0`

4. **开始处理**
   - 系统会自动:
     - 识别语音 → 翻译 → 火山TTS配音 → 上传OSS → 生成摘要 → 存储向量库

## 常见问题

### Q1: 音色列表没有显示?
**原因**: TTS类型未选择"字节火山TTS"
**解决**: 先选择TTS类型,音色列表会自动更新

### Q2: 某个音色无法使用?
**原因**:
- 该音色需要单独购买
- 账号权限不足

**解决**:
- 联系火山引擎客服购买
- 或选择其他免费音色

### Q3: Cluster ID 填错了怎么办?
**影响**: 可能导致合成失败
**解决**:
- 重新获取正确的 Cluster ID
- 或使用 `volcengine_streaming`

### Q4: 英文音色能读中文吗?
**答**: 不建议。中文内容请使用中文音色,英文内容使用英文音色。
