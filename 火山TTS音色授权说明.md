# 火山引擎 TTS 音色授权说明

## 重要说明

### 您遇到的问题

**现象**: 控制台只显示2个音色,但代码中有100+个音色选项

**原因**: 火山引擎TTS的音色需要**单独授权/购买**

### 当前可用音色

根据您的火山引擎控制台截图,您的账号目前只有:

| 音色名称 | Voice_type | 场景 |
|---------|-----------|------|
| 通用女声 | BV001_streaming | 通用场景 |
| 通用男声 | BV002_streaming | 通用场景 |

这是火山引擎的**免费基础音色**,所有账号都有。

## 解决方案

### 方案1: 只使用已有的2个免费音色 (推荐)

**优点**:
- ✅ 完全免费
- ✅ 无需额外购买
- ✅ 立即可用

**使用方法**:
1. 主界面选择: **TTS类型** = `字节火山TTS`
2. **配音角色**下拉列表会显示所有音色(100+)
3. **只选择**:
   - **通用女声** (BV001_streaming)
   - **通用男声** (BV002_streaming)
4. 其他音色如果选择了,会导致403错误

**适用场景**:
- 新闻资讯
- 教育培训
- 一般视频配音

### 方案2: 购买更多音色授权

如果您需要其他音色(如灿灿、擎苍、方言等):

#### 步骤:

1. **访问火山引擎控制台**:
   ```
   https://console.volcengine.com/speech/app
   ```

2. **进入"音色管理"**

3. **查看音色列表**:
   - 免费音色: BV001_streaming, BV002_streaming
   - 付费音色: 标有价格标签

4. **购买授权**:
   - 点击想要的音色
   - 查看定价
   - 购买授权

5. **授权后**:
   - 返回应用,音色会自动可用
   - 在主界面可以选择新购买的音色

#### 常见付费音色价格参考:

| 音色类型 | 大致价格 |
|---------|---------|
| 通用进阶音色(灿灿2.0等) | ¥500-1000/年 |
| 情感音色 | ¥300-800/年 |
| 方言音色 | ¥200-500/年 |
| 儿童/卡通音色 | ¥300-600/年 |

*具体价格以火山引擎官网为准*

## 常见问题

### Q1: 为什么选择其他火山音色会403错误?
**A**: 因为这些音色需要单独购买授权,您的账号没有这些音色的使用权限。

### Q2: 免费的通用女声/男声音质如何?
**A**: 音质很好,适合大部分场景。如果需要特殊音色(如方言、儿童音等)才需要购买。

### Q3: Edge TTS 和火山TTS哪个更好?
**A**:
- **Edge TTS**: 免费,音色多,偶尔失效
- **火山TTS**: 需付费,更稳定,商业推荐

### Q4: 可以混用吗?
**A**: 可以!不同视频可以选择不同的TTS服务。
